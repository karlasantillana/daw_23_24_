git init (inicia el archivo git) --> crear un repositorio nuevo o reiniciar uno existente
o
git clone [url] --> en la máquina donde quiero clonar


//En la máquina donde quiero subir los cambios
--> git status
--> git add . (selecciona todos los archivos que hayan cambiado)
--> git commit -m "Primera subida" guarda la seleccion
--> git push --> actualizar cambios (si es el primer push --set-upstream [url])


//en la máquina donde quiero actualizar
--> git pull -->lo abro en la consola directamente en la carpeta de la máquina donde falta actualizar
